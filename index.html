Two-stack Pushdown Automata simulator

Syntax

<state_name> <character_read> <stack0_top> <stack1_top> <next_state> <stack0_push_symbol> <stack1_push_symbol> <is_start_state> <is_end_state>

Put spaces after every element
For is_start_state, put "*" if specified state is a start state and "_" otherwise
For is_end_state, put "&" if specified state is an end state and "_" otherwise
